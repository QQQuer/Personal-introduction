个人介绍网站

这是一个现代化的个人介绍网站，展示了我的教育背景、项目经历和成就荣誉。

## 🌟 特性

- **响应式设计** - 完美适配桌面端和移动端
- **现代化UI** - 使用 Tailwind CSS 构建美观的界面
- **流畅动画** - 丰富的交互效果和过渡动画
- **单页应用** - 平滑滚动和导航体验
- **主题切换** - 支持明暗主题切换
- **联系表单** - 集成消息发送功能

## 🚀 技术栈

- **HTML5** - 语义化标记
- **CSS3** - Tailwind CSS + 自定义样式
- **JavaScript** - 原生 ES6+ 
- **Font Awesome** - 图标库
- **响应式设计** - 移动优先

## 📁 项目结构

```
Personal-introduction/
```

## 📱 响应式设计

网站采用移动优先的设计理念，确保在各种设备上都有良好的体验：

- **手机端** (< 768px) - 单列布局，汉堡菜单
- **平板端** (768px - 1024px) - 两列布局，完整导航
- **桌面端** (> 1024px) - 多列布局，丰富交互

## 🛠️ 安装和运行

### 本地运行
1. 克隆项目到本地
```bash
git clone https://github.com/QQQuer/Personal-introduction.git
```

2. 进入项目目录
```bash
cd Personal-introduction
```

3. 直接在浏览器中打开 `index.html`

### 使用 Live Server
推荐使用 VS Code 的 Live Server 扩展进行开发：
1. 安装 Live Server 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 📝 自定义配置

### 修改个人信息
编辑 `index.html` 中的以下部分：
- 姓名和标题

### 修改样式

### 修改功能
编辑 `script.js` 来调整：

- 动画逻辑
- 主题设置
